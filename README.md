# React Tasks app

A minimalistic web app for your daily agendas. A fullstack javascript/typescript project.


### Tech Stack

Frontend :-
- HTML/CSS
- React JS
- Typescript

Server :-
- Node JS
- Express JS

Database :-
- MySql
